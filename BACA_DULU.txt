┌───────────────────────────────────────────────────────────────────────────────┐
│ 🚀 CARA DEPLOY KE VERCEL (10 MENIT)                                          │
└───────────────────────────────────────────────────────────────────────────────┘

  LANGKAH 1: Push ke GitHub
    git add .
    git commit -m "Ready for deployment"
    git push

  LANGKAH 2: Deploy di Vercel
    1. Buka https://vercel.com
    2. Login dengan GitHub
    3. Klik "Add New" → "Project"
    4. Import repository kamu
    5. Set environment variables:
       - SUPABASE_URL
       - SUPABASE_KEY
       - SUPABASE_SERVICE_KEY
       - NODE_ENV=production
    6. Klik "Deploy"
    7. Tunggu 2-5 menit

  LANGKAH 3: Test Production
    1. Buka URL Vercel (https://your-project.vercel.app)
    2. Register akun baru
    3. Test chest game, wheel game, crash game
    4. Check inventory
    5. Test admin panel (jika admin)

  ✅ SELESAI! Website live! 🎉
