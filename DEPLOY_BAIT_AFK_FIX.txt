✅ HOW PERSISTENT AFK WORKS:

1. User clicks "Start AFK"
2. Backend creates session in afk_fishing_sessions
3. User closes browser / logs out
4. Session persists in database
5. Background worker processes active sessions
6. User logs in again
7. Frontend checks /afk/status
8. Shows: "AFK active, Duration: 45 minutes"
9. User clicks "Stop AFK"
10. Session deactivated
11. Fish caught, bait consumed

KEY FEATURE: Browser dapat ditutup, AFK tetap jalan!
