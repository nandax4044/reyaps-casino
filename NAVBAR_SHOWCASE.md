## 🎬 ANIMATIONS & INTERACTIONS

### Scroll Animation
```
User scrolls down
  ↓
At 100px: Transition starts
  ↓
300ms smooth animation
  ↓
Background changes to white
  ↓
Text color changes to gray
  ↓
Active link color changes to blue
```

### Mobile Menu Animation
```
User clicks hamburger
  ↓
Menu slides in from right (300ms)
  ↓
Backdrop fades in
  ↓
User can interact with menu
  ↓
Click link or outside
  ↓
Menu slides out (300ms)
  ↓
Backdrop fades out
```

### Link Hover Effect
```
User hovers over link
  ↓
Color smoothly transitions (300ms)
  ↓
Visual feedback provided
  ↓
User knows link is interactive
```
