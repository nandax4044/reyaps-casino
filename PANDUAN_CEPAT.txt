LANGKAH 1: BUKA SUPABASE
---------------------------
1. Buka browser
2. Pergi ke: https://supabase.com
3. Login dengan akun Anda
4. Pilih project Anda
5. Klik "SQL Editor" di menu kiri

LANGKAH 2: JALANKAN SQL
---------------------------
1. Buka file: JALANKAN_INI.sql
2. Tekan Ctrl+A (select all)
3. Tekan Ctrl+C (copy)
4. Kembali ke Supabase SQL Editor
5. Tekan Ctrl+V (paste)
6. Klik tombol "RUN" (atau tekan Ctrl+Enter)
7. Tunggu sampai selesai
